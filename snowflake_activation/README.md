# KIT D'ACTIVATION SNOWFLAKE
**Flight Delay Prediction System - Datascientest 2026**

## CONTENU DU KIT

Ce kit contient tout pour activer Snowflake et migrer vos données MongoDB en 30min-2h.

### Fichiers

| Fichier | Description | Durée |
|---------|-------------|-------|
| 00_test_snowflake_connection.py | Test connexion | 2 min |
| 01_snowflake_setup.sql | Création structure | 5 min |
| 02_migrate_to_snowflake.py | Migration MongoDB → Snowflake | 30min-2h |
| 03_sync_mongodb_to_snowflake.py | Sync continue (optionnel) | 5-10 min |
| ACTIVATION_GUIDE.md | Guide complet | - |

## UTILISATION RAPIDE

1. Activer Snowflake (signup.snowflake.com)
2. Configurer .env
3. Tester : python 00_test_snowflake_connection.py
4. Créer structure : 01_snowflake_setup.sql (via Web UI)
5. Migrer : python 02_migrate_to_snowflake.py

## DOCUMENTATION

Voir ACTIVATION_GUIDE.md pour le guide détaillé.
